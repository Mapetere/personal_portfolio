LatterPay

A production-grade WhatsApp bot for managing donations and payments with end-to-end encryption, resilience patterns, and comprehensive error handling.

Python 3.9+ Flask WhatsApp API Paynow Docker Terraform

The Problem

Organizations in Zimbabwe needed a simple, accessible way to collect donations via mobile money. WhatsApp is the most popular messaging platform, and mobile money (EcoCash, OneMoney) is the primary payment method. LatterPay bridges these two, creating a seamless donation experience.

Key Features

WhatsApp Integration

Full WhatsApp Business API integration via Pygwan. Supports text messages, interactive buttons, and list menus.

Payment Processing

Paynow integration supporting EcoCash, OneMoney, TeleCash, and USD payments with real-time status updates.

End-to-End Encryption

Meta Flow encryption with AES-256-CBC and RSA for secure data exchange.

Circuit Breaker

Prevents cascading failures when external services are down. Auto-recovers after 60 seconds.

Rate Limiting

Token bucket algorithm protects against abuse with 30 requests per user limit.

Analytics Dashboard

Admin panel with PDF/Excel report generation, payment statistics, and daily summaries.

Deployment

LatterPay can be deployed using Docker, Railway, Heroku, or Terraform:

# Docker
docker build -t latterpay .
docker run -p 8010:8010 --env-file .env latterpay

# Railway
railway login && railway deploy

# Terraform
cd terraform && terraform init && terraform apply
