Axion Sync

A privacy focused cycle tracking application with a clean medical grade interface and secure data management built for peace of mind.

React Supabase Tailwind CSS Vite

The Philosophy

In an era where personal data is often treated as a commodity, Axion Sync stands as a testament to the belief that privacy is a fundamental human right. I built this application to provide a sacred, secure space for health tracking, ensuring that sensitive information remains exactly where it belongs: in the hands of the user. It's not just a tracker; it's a sanctuary of digital peace.

Security as a Caretakers's Duty

The technical heartbeat of Axion Sync is its local-first architectural pattern. By employing AES-256 encryption at the edge and ensuring data is synchronized only via secure, encrypted tunnels, I've crafted a system that prioritizes user safety above all else. This isn't just about defending against threats; it's about honoring the trust users place in the technology I build.

Key Features

Secure Data

End to end encryption for all personal health data, ensuring that only you can access your most sensitive information.

Clean Interface

A minimized, distraction free UI that focuses on clarity and ease of use for tracking daily observations.

Flexible Cycles

Adaptable algorithms that account for cycle variations. providing more accurate predictions over time.

Private Sync

Seamless synchronization across devices using secure, privacy focused backend infrastructure.

Technical Implementation

  • Frontend: Built with React and Vite for optimal performance and a smooth user experience.
  • Backend: Powered by Supabase for secure authentication and real-time database synchronization.
  • Styling: Utilizes Tailwind CSS for a highly responsive and medical grade minimalist design.
