Hanaka Hardware

A comprehensive e-commerce platform for local hardware stores with real-time inventory management and Paynow integration.

React 18 Supabase Paynow Tailwind CSS Real-time DB

Vision

Hanaka Hardware was designed to centralize and modernize the hardware shopping experience in Zimbabwe. Most hardware stores operate with siloed inventory systems, leading to stock discrepancies. This project provides a unified storefront with live stock updates across branch locations.

Key Challenges

Real-time Synchronization

Maintaining accurate stock levels across multiple branch locations while users are making concurrent purchases was the biggest technical hurdle. Overselling a single unit can lead to poor customer satisfaction and operational friction.

The Solution

I implemented Supabase Postgres triggers and real-time subscriptions. Every time a checkout process begins, the system locks the specific inventory record and uses a broadcast channel to update all connected clients instantly, ensuring a single source of truth.

Key Features

Smart Inventory

Dynamic stock tracking with automatic low-stock alerts for branch managers.

Payment Integration

Full Paynow support for EcoCash, OneMoney, and Visa/Mastercard payments.

Multi-Branch Support

Ability for users to select their nearest branch and check availability locally.

Admin Dashboard

Real-time sales analytics and inventory management for the management team.
