# Nyasha Mapetere - Personal Portfolio

A modern, responsive portfolio website showcasing my projects and skills as a software engineer.

## Live Demo

Open `index.html` in your browser to view the portfolio locally.

## Features

- **Modern Design** - Dark theme with gradient accents and smooth animations
- **Responsive** - Works on desktop, tablet, and mobile devices
- **Interactive** - Typewriter effect, scroll animations, and hover states
- **Project Showcase** - Detailed project pages with documentation

## Structure

```
personal_portfolio/
├── index.html          # Main portfolio page
├── styles.css          # All styling
├── script.js           # Animations and interactivity
├── projects/
│   └── latterpay.html  # LatterPay project detail page
└── README.md
```
